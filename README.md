# Eddie Youman Services — Website

Next.js marketing site with online booking, contact forms, and admin dashboard for [Eddie Youman Services](https://eddie.kartersanamo.com) — construction clean up in Jacksonville, FL.

See [`web/README.md`](web/README.md) for local development, deployment, customer flow, and production ops (Stripe webhook, cron jobs, smoke tests).

## Quick start

```bash
cd web
cp .env.example .env
./deploy.sh
```

Public URL: https://eddie.kartersanamo.com
